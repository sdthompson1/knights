kts.PopUpWindow

Name

kts.PopUpWindow – display a tutorial window

Synopsis

kts.PopUpWindow(params)

Description

This is used during the Tutorial to show a tutorial/help message to the player.

params is a table containing the following fields:

  • title is a string giving the pop-up window title.
  • body is a string giving the main message to show in the window.
  • popup is a boolean value controlling whether this is a "foreground" or "background" window.
  • graphics is a table of graphics to show.
  • colour_changes allows the graphics to be re-coloured if needed.

Return Value

No value is returned.
