kts.RandomRange

Name

kts.RandomRange – generate a random number between two given values

Synopsis

result = kts.RandomRange(low, high)

Description

Generates a random integer between low and high inclusive.

Return Value

The generated number is returned.

Errors

If the two input values are not integers then an error is raised.

Bugs

If high is less than low then you might get unpredictable results.
