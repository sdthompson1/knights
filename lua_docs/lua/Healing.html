kts.Healing

Name

kts.Healing – restore the current actor to maximum health

Synopsis

kts.Healing(message)

Description

If the current actor (as determined by cxt.actor) is a valid creature, their current hitpoints are immediately restored to maximum.

If the current actor is a knight, then additionally, the given message is flashed on screen above the knight a number of times.

Return Value

None.
